Análisis de Reclamaciones de Seguros de Auto

Exploración de datos y hallazgos clave sobre patrones de reclamaciones

Resumen del Proyecto

Este análisis explora un conjunto de datos de reclamaciones de seguros de automóviles para identificar patrones y factores asociados con la probabilidad de que un asegurado presente una reclamación.

Dataset: Car Insurance Claim

El conjunto de datos incluye información sobre pólizas de seguro de automóviles con las siguientes variables clave:

Variable Descripción

Distribución inicial de las variables principales

Limpieza y Preparación de Datos

Se realizaron las siguientes transformaciones:

  • Normalización de edades (de proporciones a años enteros)
  • Conversión de policy_tenure a meses
  • División de max_power en caballos de fuerza y RPM
  • Detección y manejo de valores atípicos

Datos Interesantes

  • No se encontraron valores nulos en el dataset original
  • El 6.4% de los asegurados presentaron reclamaciones (dataset desbalanceado)
  • Se identificaron y manejaron valores atípicos en varias variables

Hallazgos Clave

Principales Patrones Identificados

  • Vehículos más antiguos tienden a presentar más reclamaciones
  • Vehículos con menos airbags (ej. 2) tuvieron más reclamaciones
  • Motores de 4 cilindros mostraron mayor asociación con reclamaciones
  • Clientes en áreas de menor densidad poblacional presentaron más reclamos
  • Pólizas con menor antigüedad (<6 meses) tuvieron mayor siniestralidad
